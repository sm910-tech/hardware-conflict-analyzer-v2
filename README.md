# Hardware Conflict Analyzer

Hardware Conflict Analyzer is a web-based tool that helps users check PC hardware compatibility and identify potential conflicts between components.

## Features

* CPU compatibility checking
* GPU compatibility checking
* RAM compatibility checking
* Storage compatibility checking
* Hardware database system
* Fast conflict detection
* Simple and responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run a local web server.
4. Open the provided localhost URL in your browser.
